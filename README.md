## 🌌 Overview
**OSN (Global Intelligence)** is a unified security analysis framework designed to process global threat vectors and generate high-fidelity, command-center-style intelligence reports. The system integrates data processing, automated unit testing, and futuristic visualization into a single, high-performance execution flow.

## 🚀 Key Features
- **NASA-Style Command Center UI:** A futuristic, dark-mode HTML report featuring Glassmorphism, neon-glow indicators, and real-time scanline effects.
- **Unified Architecture:** Application logic and testing suite combined into a single robust module (`analysis.py`) to eliminate environment-related import errors.
- **Automated CI/CD:** Integrated with GitHub Actions for manual and push-triggered analysis cycles.
- **Zero-Dependency Core:** Utilizes standard Python libraries for maximum compatibility and speed.

## 🛠️ System Architecture
The system follows a three-stage execution pipeline:
1. **Intelligence Fetching:** Simulates or retrieves raw security data from global nodes.
2. **Analysis Engine:** Processes threat levels (Critical, High, Medium, Low) and calculates system-wide alert statuses (ALPHA-RED, etc.).
3. **Visualization Layer:** Generates a professional `index.html` report and automatically deploys it for review.
