## ⚡️ Project Overview

**Global Security Data Poll** is a massive, highly-classified-styled intelligence aggregator. It actively pipelines raw RSS feeds from global security networks and structures historical operations into an interactive 3D command environment. 

Developed as a cutting-edge technical portfolio piece, it bridges the gap between massive data engineering and incredibly fluid UI/UX design.

> **⚠️ REPOSITORY NOTICE:**
> To protect the proprietary architecture of the overall platform and prevent 1:1 cloning of the live site, **this repository serves solely as a Technical Showcase**. It contains the core, hand-crafted files that demonstrate my engineering capabilities. The full Next.js site skeleton remains private.

---

## 💻 Technical Showcase Highlights

Inside this repository, you will find the critical engine files that power the live site. They demonstrate my ability to handle complex data, build beautiful 3D experiences, and maintain high-performance frontend code.

### 🌐 1. `GlobeMap.tsx` | Interactive 3D Threat Map
The crown jewel of the platform. A fully interactive, WebGL-powered 3D globe visualization utilizing WebXR and modern rendering techniques.
*   **Tactical Glassmorphism:** Custom-built HTML overlay markers that track to 3D coordinates.
*   **Performance First:** Renders flawless 60FPS rotational mechanics even when loaded with global data points.
*   **Radar Pings:** Custom CSS keyframe animations for high-end "sonar/radar" blip effects on hotspots.

### 🗄️ 2. `CalendarArchive.tsx` | Historical Intel Grid
A massive display of React layout mastery utilizing `framer-motion`.
*   **Dynamic Grid:** Implements a heavily customized fluid grid layout (`grid-cols-1 lg:grid-cols-2`) that dynamically scales massive data.
*   **Agentic Search:** Deep filtering logic that parses across titles, bodies, and tags seamlessly.
*   **Micro-interactions:** Hover states, accent lines, and animated transitions to make static data feel alive.

### 📡 3. `rss.ts` | The Data Pipeline
The brain behind the OSINT bot aggregator.
*   **Asynchronous Fetching:** Robust fetching, parsing, and normalization of diverse RSS schemas (Israel Hayom, Naval News, DW, etc.).
*   **Fault Tolerance:** Clean processing loops and fallback parameters for high-velocity data ingestion.

### 🧬 4. `archive.json` | Intelligence Database
A highly structured JSON database representing over 80 major global security and intelligence events throughout history—from the deciphering of the Enigma code up to the recent Blue Ghost Lunar Landing. This powers the entirety of the Calendar/Archive view.
